### C. **Project-Based Learning with Portfolio Builder**
**What:** Every course includes real-world projects that build student portfolios

**Why It Works:**

- Increases course completion (students want to finish portfolio)

**Features:**

- Public portfolio URLs: `devacademy.com/portfolio/username`
- GitHub integration for code projects
- LinkedIn-shareable certificates with project showcase

**Database Schema:**
```sql
CREATE TABLE student_portfolios (
  id INT AUTO_INCREMENT PRIMARY KEY,
  client_id INT NOT NULL,
  portfolio_url VARCHAR(255) UNIQUE,
  is_public TINYINT(1) DEFAULT 1,
  custom_domain VARCHAR(255),
  theme VARCHAR(50) DEFAULT 'modern',
  bio TEXT,
  skills JSON,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (client_id) REFERENCES client(id)
);

CREATE TABLE portfolio_projects (
  id INT AUTO_INCREMENT PRIMARY KEY,
  portfolio_id INT NOT NULL,
  course_id INT NOT NULL,
  project_title VARCHAR(200),
  project_description TEXT,
  demo_url VARCHAR(255),
  github_url VARCHAR(255),
  image_url VARCHAR(255),
  technologies JSON,
  completed_at TIMESTAMP,
  FOREIGN KEY (portfolio_id) REFERENCES student_portfolios(id),
  FOREIGN KEY (course_id) REFERENCES courses(course_id)
);
```

**Marketing Hook:**
> "Learn to code AND build a professional portfolio at the same time!"
